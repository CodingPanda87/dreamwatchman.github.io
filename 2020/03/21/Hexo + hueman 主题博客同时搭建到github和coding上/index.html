Hexo使用

  • $ hexo new post helloHexo ——- 新建博客页面
  • $ hexo clean —————————– 清除历史版本
  • $ hexo g 或 $ hexo generate ——– 生成静态网页
  • $ hexo s或$ hexo server ————– 启动服务预览
  • $ hexo d或$ hexo deploy ————– 上传部署站点
